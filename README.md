## Code

**Voici un code qui permet de _print_ 10 fois le mot <ins> "Bonjour !" </ins>**
```
for i in range(10): # de 1 à 10
  print("Bonjour !")
```
**Ou bien il est également possible de cette façon**
```
for i in range(1, 11): # de 1 à 10
  print("Bonjour !")
```
